# MyClawX 职业场景自定义配置系统设计方案

## 一、系统概述

### 1.1 设计目标
在原有职业预设系统基础上，增加**用户自定义创建**能力，让用户能够：
- 创建全新的职业
- 在现有职业下创建自定义场景
- 克隆并修改官方配置
- 分享和导入他人配置

### 1.2 核心价值
- **个性化定制** - 满足特殊行业和个性化需求
- **知识共享** - 社区驱动的配置生态
- **快速迭代** - 基于现有配置快速调整
- **低门槛创建** - 可视化编辑器，无需编写 JSON

### 2.2 配置继承与覆盖机制

```typescript
// 配置合并优先级（从高到低）
const configPriority = [
  'runtimeOverrides',     // 运行时用户调整
  'userCustom',           // 用户自定义配置
  'communityImported',    // 社区导入配置
  'officialPresets',      // 官方预设
];

// 合并策略
interface ConfigMergeStrategy {
  // 对象类型：深度合并
  skills: 'deepMerge';
  config: 'deepMerge';
  
  // 数组类型：替换或追加
  cronTasks: 'replace' | 'append';
  promptTemplates: 'replace' | 'append';
  
  // 标量类型：高优先级覆盖
  name: 'override';
  description: 'override';
}
```

## 三、数据模型扩展

### 3.1 扩展的职业定义

```typescript
// src/types/profession.ts

export interface Profession {
  // ... 原有字段
  
  // 新增：配置来源信息
  source: {
    type: 'official' | 'custom' | 'community' | 'cloned';
    version: string;
    author?: string;
    authorId?: string;
    createdAt: string;
    updatedAt: string;
    originalId?: string;      // 克隆来源ID
    importUrl?: string;       // 社区导入来源
  };
  
  // 新增：权限控制
  permissions: {
    editable: boolean;
    deletable: boolean;
    shareable: boolean;
    cloneable: boolean;
  };
  
  // 新增：版本控制
  version: string;
  changelog?: string[];
  
  // 新增：标签和分类
  category?: string;          // 职业分类
  tags: string[];
  keywords?: string[];        // 搜索关键词
}

// 用户自定义职业元数据
export interface CustomProfessionMeta {
  id: string;
  isPublished: boolean;
  publishVersion?: string;
  shareCode?: string;         // 分享码
  shareUrl?: string;
  downloadCount?: number;
  rating?: number;
}
```

```typescript
async function cloneProfession(
  sourceId: string, 
  options: CloneOptions
): Promise<Profession> {
  // 1. 获取源配置
  const source = await getProfession(sourceId);
  
  // 2. 创建新配置
  const cloned: Profession = {
    ...source,
    id: generateId(),
    name: `${source.name} (副本)`,
    source: {
      type: 'cloned',
      originalId: sourceId,
      createdAt: new Date().toISOString(),
      updatedAt: new Date().toISOString(),
    },
    permissions: {
      editable: true,
      deletable: true,
      shareable: true,
      cloneable: true,
    },
  };
  
  // 3. 保存到用户自定义目录
  await saveCustomProfession(cloned);
  
  // 4. 打开编辑器
  return cloned;
}
```

## 九、文件格式与存储

### 9.1 导出文件格式

```typescript
// 导出文件结构
interface ExportedProfession {
  format: 'clawx-profession';
  version: '1.0.0';
  exportedAt: string;
  exportedBy?: string;
  
  profession: Profession;
  
  // 包含的额外数据
  includes: {
    skillConfigs: boolean;
    apiKeys: boolean;        // 加密存储
    personalData: boolean;
  };
  
  // 签名（用于验证完整性）
  signature?: string;
}

// 文件扩展名: .clawxprof
// MIME type: application/x-clawx-profession
```

## 十一、总结

通过这套自定义配置系统，用户可以：

1. **创建全新职业** - 从零开始定义职业、场景、技能组合
2. **克隆修改** - 基于官方或社区配置快速定制
3. **可视化编辑** - 无需编写 JSON，通过表单和拖拽完成配置
4. **分享导入** - 通过分享码、JSON、文件等方式交换配置
5. **测试验证** - 在应用前测试配置是否正常工作

这将形成一个**用户驱动的配置生态**，让 MyClawX 能够覆盖更多行业和场景。
